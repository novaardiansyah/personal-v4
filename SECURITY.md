# Security Policy

The following versions of this project are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Nova Ardiansyah via [novaardiansyah78@gmail.com](mailto:novaardiansyah78@gmail.com). All security vulnerabilities will be promptly addressed.
